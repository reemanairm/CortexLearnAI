# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# CortexLearn AI

CortexLearn AI is an AI-powered learning assistant that converts study materials into interactive learning resources such as flashcards, quizzes, and contextual explanations.

Users can upload study documents and interact with them using AI to improve understanding, revision, and learning efficiency.

---

## Features

### Document Upload
- Upload study materials in **PDF format**
- Built-in **embedded PDF viewer**

### AI Flashcard Generation
- Automatically generates flashcards from uploaded documents
- Customizable **number of flashcards**
- Each flashcard includes **Ask AI** to clarify concepts

### AI Quiz Generation
- Generate quizzes from uploaded study materials
- Customizable **difficulty levels**
- Adjustable **number of questions**
- Each quiz question includes **AI assistance for doubts**

### AI Document Chat
- Ask questions directly about the uploaded document
- Context-aware AI responses

### Authentication
- Secure **OAuth-based login system**

### Progress Tracking
- Tracks quiz performance and learning progress

### Admin Dashboard
- Monitor application activity
- View user issues
- Mark support requests as resolved

### Help System
- Users can report issues
- Admin can manage and resolve them

---

## Installation

Clone the repository

git clone https://github.com/reemanairm/CortexLearnAI.git⁠

Navigate to the project directory

cd cortexlearn-ai

Install dependencies

npm install

Run development server

npm run dev
